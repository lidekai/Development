## 下载代码 <a name="node.js-01" href="#" >:house:</a>

> 运行git bash here，使用命令

```
git clone https://github.com/elunez/eladmin-web.git
```

## 配置 <a name="node.js-02" href="#" >:house:</a>

> 下载安装组件

- vs code 安装快捷键<kbd>Ctrl</kbd>+<kbd>`</kbd> 进入终端，输入命令

```
cnpm install
```

> 运行

```
cnpm run dev
```
